Investrs Inn

A web-based investment and trading platform that allows users to buy and sell financial assets including stocks, forex, and indices. The system provides user registration, login, dedicated trading pages for each asset type, and secure database storage of user information. Built using PHP, MySQL, HTML, CSS, and JavaScript.
